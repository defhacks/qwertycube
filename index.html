QWERTYcube - Keyboard oriented Rubik's Cube

by Steven Elliott based on rubik_scrambler by Siddhante Nangla

Overview

As the name suggests QWERTYcube was originally intended to be a keyboard based cube. However, it has evolved to have mouse support as well as touch support on mobile devices. If you just want to see a reference describing the buttons and keys scroll down.

Keyboard

The keyboard can be used to either make moves in the expected way (R, L, U, D, etc.) or for various commands (T, L, etc.). Some commands that destroy information, such as N and P, require that Alt and Shift be pressed at the same time. For moves the Alt and Shift make the move double layer and inverse respectively. All three middle moves and entire cube rotations (x, y and z) are supported. The keys can be reconfigured (see "Settings").

Mouse And Touch

Mouse and touch support should be intuitive, but there are some things about it that may not be obvious. Clicking on the grey background rotates the entire cube. Note that when the cube is rotated this way the R, L, U, D, etc. labels follow the cube (see "Orientation labels"). If rotation lock (see "Rotation lock") is on the program does it's best to interpret movement on the grey background as being a cube move.

Clicking the mouse on the cube and moving results in a cube move. Although it should be obvious how this works in the case of moving a single layer it's also possible to move multiple layers, such as a double layer move (r, etc.) or even the entire cube (x, etc.). To move multiple layers the swipe should cross a layer boundary. All touched layers will rotate.

Settings

Pressing the Settings button displays the settings dialog. This is where various obscure and advanced settings are stored. Read the text at the start of the settings dialog for an overview of how settings work. Click Ok to store the settings to persistent storage. Note that persistent storage can be cleared via the P key.

Key Tables

Control Key Button Description
A Toggle animation
C Savepoint Set a savepoint. Try with Alt-G
G Undo Undo last move. Shift to redo
H Help Display this help message
I Settings Display the settings
J Scramble Scramble the cube. By default jsss is used.
K Lock Rotation lock. Prevent cube rotations.
N New New solved cube
O Toggle display orientation
P Reset Persistent storage clear (factory reset)
T Timer Toggle the timer

Face Key Description
R Rotate the right face clockwise
L Rotate the left face clockwise
U Rotate the upper face clockwise
D Rotate the down face clockwise
F Rotate the front face clockwise
B Rotate the back face clockwise

Center Key Description
M Middle turn around the X-axis
E Equator turn around the Y-axis
S Slice turn around the Z-axis

Axis Key Description
X Rotate entire cube about the X-axis
Y Rotate entire cube about the Y-axis
Z Rotate entire cube about the Z-axis

Modifier Key Description
Shift Anti-clockwise instead of clockwise
Alt Turn two layers instead of one
Esc Alternative to Alt
